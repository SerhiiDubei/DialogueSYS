# Інструкція з використання AI Assistant Hub

## Що це таке?

**AI Assistant Hub** - це аддон для Godot, який дозволяє використовувати AI асистентів (як ChatGPT, Ollama, Gemini тощо) прямо в редакторі Godot. Ви можете спілкуватися з AI, щоб отримати допомогу з кодуванням, створенням діалогів, налаштуванням сцен та іншим.

## Встановлення

✅ Аддон вже встановлено в `addons/ai_assistant_hub/` та додано до списку увімкнених плагінів у `project.godot`.

**Щоб активувати аддон:**
1. Відкрийте проект у Godot
2. Перейдіть в меню: **Project → Project Settings → Plugins**
3. Знайдіть **"AI Assistant Hub"** та увімкніть його (поставте галочку)
4. Перезапустіть Godot (закрийте та відкрийте знову)

## Як відкрити AI Hub

Після активації плагіна внизу редактора з'явиться вкладка **"AI Hub"**. Натисніть на неї, щоб відкрити панель AI асистента.

## Налаштування LLM провайдера

AI Assistant Hub підтримує кілька провайдерів AI:

### 1. **Ollama** (рекомендовано для початку)
- **Що це:** Локальний AI, який працює на вашому комп'ютері
- **Переваги:** Безкоштовно, не потрібен API ключ, працює офлайн
- **Як налаштувати:**
  1. Завантажте та встановіть [Ollama](https://ollama.ai/)
  2. Запустіть Ollama
  3. Завантажте модель (наприклад, `ollama pull llama2` або `ollama pull mistral`)
  4. В AI Hub виберіть провайдер **"Ollama"**
  5. Server URL залиште як `http://localhost:11434` (за замовчуванням)
  6. Натисніть **"Refresh Models"** - ви побачите список завантажених моделей

### 2. **Google Gemini**
- **Що це:** AI від Google
- **Як налаштувати:**
  1. Виберіть провайдер **"Gemini"**
  2. Отримайте API ключ на [Google AI Studio](https://makersuite.google.com/app/apikey)
  3. Вставте API ключ у поле "API key"
  4. Натисніть **"Refresh Models"**

### 3. **OpenRouter**
- **Що це:** Платформа з доступом до багатьох AI моделей (GPT-4, Claude тощо)
- **Як налаштувати:**
  1. Виберіть провайдер **"OpenRouter"**
  2. Отримайте API ключ на [OpenRouter.ai](https://openrouter.ai/)
  3. Вставте API ключ
  4. Натисніть **"Refresh Models"**

### 4. **OpenWebUI**
- **Що це:** Веб-інтерфейс для Ollama
- **Як налаштувати:**
  1. Встановіть OpenWebUI (зазвичай працює на `http://localhost:3000`)
  2. Виберіть провайдер **"OpenWebUI"**
  3. Вставте URL вашого OpenWebUI сервера
  4. Натисніть **"Refresh Models"**

## Створення AI асистента

Після налаштування провайдера:

1. **Виберіть модель** зі списку доступних моделей
2. Натисніть кнопку **"New assistant type"**
3. Заповніть форму:
   - **Name** - назва асистента (наприклад, "GDScript Helper")
   - **Description** - опис (необов'язково)
   - **System Prompt** - інструкції для AI (наприклад, "Ти експерт з GDScript та Godot. Допомагай з кодом українською мовою.")
   - **Icon** - іконка асистента (необов'язково)
4. Натисніть **"Create"**

## Використання AI асистента

1. **Відкрийте AI Hub** (вкладка внизу редактора)
2. **Натисніть на кнопку асистента**, якого ви створили
3. **Починайте спілкування** - пишіть запитання або прохання
4. AI відповість у чаті

### Приклади запитів:

- "Допоможи написати функцію для діалогової системи"
- "Як створити сцену з персонажами в Godot?"
- "Поясни, як працює Dialogue Manager"
- "Напиши код для обробки кліків миші"
- "Як змінити колір UI елементів?"

## Особливості

- **Збереження чатів:** Можна зберегти розмову для подальшого використання
- **Кілька асистентів:** Можна створити різних асистентів для різних задач
- **Інтеграція з редактором:** AI може читати та писати код у Godot

## Налаштування (Preferences)

У **Project Settings → Plugins → AI Assistant Hub** можна налаштувати:
- Куди виводити "thinking" процес AI
- Автоматично прокручувати до низу чату
- Пропускати привітання

## Підтримка

Якщо виникли проблеми:
- Перевірте, чи правильно налаштований провайдер (URL, API ключ)
- Переконайтеся, що модель завантажена (для Ollama)
- Перезапустіть Godot після активації плагіна
- Перевірте консоль на помилки
